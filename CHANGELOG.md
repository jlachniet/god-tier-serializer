# Changelog

## 0.3.0

- New supported types:
  - Function
  - Async Function
  - Generator Function
  - Async Generator Function
  - Boolean Object
  - Number Object
  - BigInt Object
- Added support for prototype serialization

## 0.2.0

- New supported types:
  - Array
  - Date
  - RegExp
  - String Object
- Migrated to webpack for building
- Added "files" field to package.json

## 0.1.0

- Initial development release
